# PRD: 家づくり価値観診断（MVP）

## 目的
注文住宅検討中の夫婦/家族が、自分たちの価値観・必要条件・予算感を入力し、最適な家づくりの方向性（30タイプ診断）を得られるWebアプリ。

## ターゲット
- 注文住宅を検討し始めた夫婦/家族
- メーカー選びの前に「自分たちが何を重視するか」を整理したい人

## 非ゴール（MVPでやらないこと）
- ユーザー認証・アカウント機能
- サーバー側データ保存
- 外部API連携（LLM等）
- メーカー名の表示・推薦
- 有料機能

## 画面一覧
| パス | 画面名 | 概要 |
|------|--------|------|
| / | LP | サービス説明、開始ボタン、免責 |
| /start | モード選択 | 単独/夫婦モード選択 |
| /questions/values | 価値観質問 | 26問（夫婦は2人分） |
| /questions/requirements | 必要条件 | 8問（世帯1回） |
| /questions/budget | 予算 | 4問（任意、スキップ可） |
| /result | 診断結果 | 30タイプ、坪数、間取り、予算ルール等 |

## 受入条件
- [ ] pnpm install → pnpm dev で起動できる
- [ ] 単独/夫婦で最後まで進めて結果が出る
- [ ] diagnose.ts の単体テストが通る（32テスト以上）
- [ ] docs が存在し、中身が埋まっている
- [ ] 免責事項がLP・結果ページに表示される
