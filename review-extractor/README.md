# 住宅メーカー口コミ抽出ツール（Google Maps 専用）

住宅メーカー名を入力すると、**Google Maps** 上の各展示場・支店の口コミを自動抽出します。

- ポジティブ口コミ（★4-5）を最大30件
- ネガティブ口コミ（★1-2）を最大30件
- 各口コミの **Google Maps 共有リンク** を自動取得

## セットアップ

```bash
cd review-extractor
npm run setup
```

> `npm run setup` は依存パッケージのインストールと Playwright ブラウザ（Chromium）のダウンロードを行います。

## 使い方

### コマンドライン引数で指定

```bash
npm run extract -- "一条工務店"
npm run extract -- "積水ハウス"
npm run extract -- "住友林業"
```

### インタラクティブモード

```bash
npm run extract
# → "住宅メーカー名を入力してください:" と表示されるので入力
```

## 出力

`output/` ディレクトリに以下の2ファイルが生成されます。

| ファイル | 形式 | 用途 |
|---------|------|------|
| `{メーカー名}_{日時}.json` | JSON | プログラムでの後処理 |
| `{メーカー名}_{日時}.tsv` | TSV | スプレッドシート貼り付け |

### TSVカラム

| カラム | 説明 |
|-------|------|
| 分類 | ポジティブ / ネガティブ |
| No. | 番号 |
| 星評価 | 1-5 |
| 投稿者 | Google Maps 上の投稿者名 |
| 日付 | 投稿日 |
| 口コミ本文 | 口コミ全文 |
| 展示場・支店名 | 口コミ対象の拠点名 |
| 口コミ共有リンク | Google Maps 上の口コミ共有URL |
| 展示場リンク | 展示場の Google Maps URL |

## 動作の仕組み

1. Google Maps で「{メーカー名} 展示場 モデルハウス」を検索
2. 検索結果の各拠点ページに移動
3. 「口コミ」タブを開き、スクロールして口コミを読み込む
4. 各口コミの三点メニュー → 共有 から共有リンクを取得
5. ★4-5をポジティブ、★1-2をネガティブに分類
6. JSON + TSV ファイルに出力

## 注意事項

- インターネット接続が必要です
- Google Maps の構造変更により動作しなくなる場合があります
- 大量リクエストでアクセス制限がかかる場合があります
- 口コミの利用は Google の利用規約に従ってください
