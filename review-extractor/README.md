# 住宅メーカー口コミ抽出ツール

住宅メーカー名を入力すると、Google Maps上の各展示場・支店の口コミを自動抽出します。
ポジティブ口コミ（★4-5）とネガティブ口コミ（★1-2）をそれぞれ最大30件抽出し、口コミの共有リンクも取得します。

## セットアップ

```bash
cd review-extractor
npm run setup
```

> `npm run setup` は依存パッケージのインストールと Playwright ブラウザ（Chromium）のダウンロードを行います。

## 使い方

### インタラクティブモード

```bash
npm run extract
```

プロンプトが表示されるので、住宅メーカー名を入力してください。

### コマンドライン引数

```bash
npm run extract -- "積水ハウス"
npm run extract -- "一条工務店"
npm run extract -- "住友林業"
```

## 出力

`output/` ディレクトリに以下の2ファイルが生成されます。

| ファイル | 形式 | 用途 |
|---------|------|------|
| `{メーカー名}_{日時}.json` | JSON | プログラムでの後処理 |
| `{メーカー名}_{日時}.tsv` | TSV | スプレッドシート貼り付け |

### TSVカラム

| カラム | 説明 |
|-------|------|
| 分類 | ポジティブ / ネガティブ |
| 星評価 | 1-5 |
| 投稿者 | 投稿者名 |
| 日付 | 投稿日 |
| 口コミ本文 | 口コミ全文 |
| 展示場・支店名 | 口コミ対象の拠点名 |
| 口コミ共有リンク | Google Maps上の口コミ共有URL |
| 展示場リンク | 展示場のGoogle Maps URL |

## 分類ロジック

- **ポジティブ**: ★4以上
- **ネガティブ**: ★2以下
- **★3（中立）**: テキスト内のキーワード分析で補助分類

## 注意事項

- Google Mapsのスクレイピングはサイトの構造変更により動作しなくなる場合があります
- 大量のリクエストを短時間で行うとアクセス制限がかかる場合があります
- 取得した口コミの利用は各サービスの利用規約に従ってください
