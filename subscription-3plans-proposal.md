# 占いサブスク × AI マネタイズ — 3プラン比較提案書

> 対象: [@uranai_terrace](https://www.instagram.com/uranai_terrace)（フォロワー約8万人）
> 前提: LINEは使わない。AIを活用し社員ゼロで運用。低コストでスケール。

### 導出した重要インサイト

1. **月額300〜500円が「痛くない」ゾーン** — 占いサブスクの成功例はすべてこの価格帯
2. **「1回だけ深く占ってほしい」需要が大きい** — 年に数回利用が7割。サブスクだけでなく都度課金も必要
3. **Instagram→外部サービスの導線は短いほど良い** — プロフィールリンク1クリックで到達できるのが理想
4. **79%が無料利用** — フリーミアム（無料体験→有料転換）が必須
5. **Instagramサブスク機能は月額50〜800円で設定可能** — プラットフォーム手数料30%（Apple/Google経由）だが、導線が最短

## プランA：Instagramサブスク特化型

### コンセプト
**「いつものInstagramの中で完結する、JUNの限定恋愛占い」**

Instagramのサブスクリプション機能（月額50〜800円）をフル活用。フォロワーがアプリを離脱せずに課金→限定コンテンツを受け取れる。

### 料金設計

| プラン | 月額 | 内容 |
|--------|------|------|
| **ベーシック** | **400円/月** | 限定ストーリーズ（毎朝の詳細パーソナライズ運勢）+ 限定リール（週1回の恋愛タロット） |
| — | — | + サブスク限定ライブ（月2回、質問に直接回答） |
| — | — | + サブスクバッジ（コメント欄で目立つ） |

※Instagramサブスクは現在1プランのみ設定可能

### 導線設計

```
[Instagramフィード/リール]
  ↓ 「続きはサブスクで」と予告
[ストーリーズで一部チラ見せ]
  ↓ 「フル版はサブスク登録者限定」
[プロフィールの「登録する」ボタン]
  ↓ ワンタップで課金完了
[限定ストーリーズ/ライブ/チャットを享受]
```

### AI活用ポイント

| 工程 | AI活用方法 |
|------|-----------|
| 毎朝の限定運勢テキスト | LLMで12星座×恋愛運を自動生成。JUNさんの文体でプロンプト設計 |
| 限定リールの台本 | AIが週間恋愛タロットの台本を自動作成 |
| ライブ時の回答補助 | ライブ中、AIが視聴者の質問に対するドラフト回答をリアルタイム提示 |
| 限定画像コンテンツ | AI画像生成で「今週の開運カード」等を自動作成 |

### 収益シミュレーション

```
フォロワー8万人
  → サブスク転換率 0.5%（控えめ）= 400人
  → 月額400円 × 400人 = 月160,000円（総売上）
  → Apple/Google手数料30% = △48,000円
  → 手取り = 月112,000円

転換率が1%に改善した場合:
  → 800人 × 400円 = 月320,000円
  → 手取り = 月224,000円

転換率が2%に改善した場合:
  → 1,600人 × 400円 = 月640,000円
  → 手取り = 月448,000円
```

### メリット

| # | メリット | 根拠 |
|---|---------|------|
| 1 | **導線が最短（ゼロクリック）** | フォロワーはInstagramアプリ内でそのまま課金。外部遷移なし |
| 2 | **開発コストゼロ** | Instagramの既存機能だけで完結。サーバー・決済実装不要 |
| 3 | **即日開始可能** | 8万フォロワーで条件クリア済み（1万人以上）。審査通過後すぐ |
| 4 | **バッジによるコミュニティ効果** | サブスク会員がコメント欄で目立つ→他のフォロワーの加入意欲を刺激 |
| 5 | **離脱率が低い** | アプリ内完結のため「面倒」「忘れた」による離脱が最小 |

### デメリット

| # | デメリット | 影響度 |
|---|----------|--------|
| 1 | **Apple/Google手数料30%** | 売上の3割を失う。月額400円なら手取り280円/人 |
| 2 | **料金の自由度が低い** | 日本では50/200/400/600/800円の固定選択肢のみ |
| 3 | **招待制で利用できない可能性** | 2023年7月以降、招待されたクリエイターのみ利用可能 |
| 4 | **プラットフォーム依存** | Instagram側の仕様変更で収益モデルが崩れるリスク |
| 5 | **都度課金商品を併売できない** | 「1回3,000円の詳細鑑定」等をInstagramサブスク内では提供不可 |
| 6 | **顧客データを自社保有できない** | メールアドレス等が取得できず、他チャネルへの展開が難しい |

## プランB：自社AI占いWebアプリ（Next.js + Stripe）

### コンセプト
**「JUN監修のAI恋愛占い」— フリーミアム型Webアプリ**

自社でWebアプリを構築。無料占い（1日1回）でユーザーを獲得し、詳細鑑定とサブスクで収益化。Stripe決済でApple/Google手数料を回避。

### 料金設計（3層構造）

| 層 | プラン名 | 料金 | 内容 |
|----|---------|------|------|
| **無料** | フリー | 0円 | 1日1回の簡易タロット（1枚引き）+ 星座別デイリー運勢 |
| **サブスク** | プレミアム | **月額480円** | 毎朝のパーソナライズ恋愛運（メール配信）+ 1日3回まで詳細タロット + 月1回のAI総合鑑定レポート |
| **都度課金** | スペシャル鑑定 | **1回1,500〜3,000円** | AI恋愛相性鑑定書（PDF 20ページ）/ 復縁可能性診断 / 年間恋愛運鑑定書 |

**なぜこの価格か**:
- 月額480円：競合のAI占いサブスク（500円）よりわずかに安く、心理的に「ワンコイン以下」
- 都度課金1,500〜3,000円：電話占い1回（4,000〜8,000円）の半額以下。「電話占いより安くて、いつでもできる」ポジション
- 調査データで「500〜9,999円の1回課金」が47.5%と最多。このゾーンを狙う

### 導線設計

```
[Instagramフィード/リール]
  ↓ 「無料で恋愛運を占う → プロフリンク」
[プロフィールのリンク]
  ↓
[自社Webアプリ：無料タロット体験]
  ↓ 結果画面で「もっと詳しく知りたい方は…」
[会員登録（メールアドレス）]
  ↓
[月額480円サブスク or 都度課金の鑑定書購入]
```

### 技術スタック・開発コスト

| 項目 | 選定 | 月額コスト |
|------|------|-----------|
| フレームワーク | **Next.js**（既存リポジトリのhome-diagnosisを転用） | 0円 |
| ホスティング | Vercel（Hobbyプラン→Proプラン） | 0〜2,000円/月 |
| AI API | Claude API / OpenAI API | 約5,000〜15,000円/月（利用量次第） |
| 決済 | **Stripe**（初期費用0円、手数料3.6%） | 従量制 |
| メール配信 | Resend / SendGrid（無料枠あり） | 0〜3,000円/月 |
| データベース | Supabase（無料枠）/ PlanetScale | 0〜3,000円/月 |
| ドメイン | 独自ドメイン | 約1,500円/年 |
| **合計固定費** | | **約5,000〜23,000円/月** |

### AI活用ポイント

| 機能 | AI活用方法 |
|------|-----------|
| タロット鑑定（チャット型） | ユーザーの質問 → LLMがJUNさんの文体でタロット解釈を生成。カード画像はAI生成 |
| パーソナライズ運勢配信 | 毎朝Cron実行。ユーザーの生年月日・星座に基づきLLMが個別運勢を自動生成→メール送信 |
| 鑑定書PDF自動生成 | 入力フォーム→LLM生成→PDF変換→自動メール送付。完全無人 |
| 相性診断 | 2人分の生年月日から LLM が詳細な相性レポートを生成。結果画面にSNSシェアボタン |

### 収益シミュレーション

```
■ サブスク収益
  フォロワー8万人 → Webアプリ訪問率10% = 8,000人
  → 無料会員登録率 30% = 2,400人
  → サブスク転換率 8% = 192人
  → 月額480円 × 192人 = 月92,160円
  → Stripe手数料3.6% = △3,318円
  → サブスク手取り = 月88,842円

■ 都度課金収益（鑑定書）
  月間アクティブ無料会員 2,400人
  → 都度課金率 3% = 72件/月
  → 平均単価 2,000円 × 72件 = 月144,000円
  → Stripe手数料3.6% = △5,184円
  → 都度課金手取り = 月138,816円

■ 合計手取り = 月227,658円
■ 固定費控除後 = 月204,658〜222,658円

＊6ヶ月後（SEO・口コミ効果込み）:
  サブスク会員500人 + 都度課金150件/月 の場合
  → 月540,000円（手取り約500,000円）
```

### メリット

| # | メリット | 根拠 |
|---|---------|------|
| 1 | **手数料が最安（3.6%）** | Instagramの30%、noteの20%と比較して圧倒的に有利 |
| 2 | **サブスク＋都度課金の二刀流** | 月額480円の安定収入 + 1回1,500〜3,000円の高単価商品を併売可能 |
| 3 | **顧客データを100%自社保有** | メールアドレス取得→メルマガ・他サービス展開が自由自在 |
| 4 | **無料占いがバイラルする** | 「結果をシェア」機能でInstagram外からもユーザー流入 |
| 5 | **完全自動化（AI + Cron + Stripe）** | 24時間無人稼働。鑑定書生成〜決済〜納品まで人間の介入ゼロ |
| 6 | **既存のNext.js基盤を転用可能** | home-diagnosisプロジェクトのコード・設計をそのまま活用 |
| 7 | **プラットフォーム非依存** | Instagramの仕様変更に左右されない自社資産 |

### デメリット

| # | デメリット | 影響度 |
|---|----------|--------|
| 1 | **開発が必要（1〜2ヶ月）** | Next.js + Stripe + AI APIの実装。ただし既存基盤転用で短縮可 |
| 2 | **月額固定費が発生** | サーバー・API費用で月5,000〜23,000円。赤字リスクは低いが存在 |
| 3 | **Instagram→外部遷移の離脱** | プロフリンク→外部サイトの遷移で20〜40%が離脱する |
| 4 | **Stripeの占い利用可否を要確認** | Stripeの禁止業種に「占い」が含まれる可能性。事前確認必須 |
| 5 | **集客がInstagram依存** | 立ち上げ初期はInstagram以外の流入が少ない（SEO効果は3〜6ヶ月後） |

## プランC：note定期購読 + STORES都度課金 ハイブリッド型

### コンセプト
**「既存プラットフォームの組み合わせで、開発ゼロ・即日開始」**

noteの定期購読マガジン（月額制）でサブスク収益を確保しつつ、既存のSTORES（鑑定書販売実績あり）で高単価商品を販売。AIで全コンテンツを効率的に生成。

### 料金設計

| チャネル | プラン | 料金 | 内容 |
|---------|--------|------|------|
| **note** | 定期購読マガジン | **月額500円** | 週1回の詳細恋愛運コラム + 月初の12星座別恋愛月間運勢 + 新月・満月の恋愛タロットリーディング + 読者限定Q&A記事（月1回） |
| **note** | 単品記事 | **300〜800円** | 「2026年下半期 星座別恋愛運」等の特別コンテンツ |
| **STORES** | AI恋愛鑑定書 | **1,500〜5,000円** | パーソナライズされた恋愛鑑定PDF（AI生成） |
| **STORES** | 年間恋愛運鑑定 | **3,000〜8,000円** | 12ヶ月分の詳細恋愛運レポート（AI生成PDF） |

**なぜこの組み合わせか**:
- noteの成功事例：月額500円×200人=月10万円は実証済み
- STORESは既に鑑定書販売の実績があり、顧客も既存
- 二つのプラットフォームで「サブスク（安定）」+「都度課金（高単価）」を実現

### 導線設計

```
[Instagramフィード/リール]
  ↓ 「詳しい恋愛運の解説はnoteで → プロフリンク」
[プロフィールのlinktree/lit.link]
  ├→ [note: 定期購読マガジン（月額500円）]
  │     └→ 記事内で「個別鑑定はSTORESで」と誘導
  └→ [STORES: AI鑑定書購入（1,500〜5,000円）]
```

| Q&A記事 | 読者からの質問をLLMがドラフト回答→JUNさんが編集 |

| # | メリット | 根拠 |

| # | デメリット | 影響度 |
|---|----------|--------|
| 1 | **手数料が高い（note25%）** | Stripe 3.6%、Instagram 30%の中間。利益率はプランBに劣る |
| 2 | **プラットフォーム二つの運用** | noteとSTORES両方の管理が必要。ただし両方とも運用負荷は低い |

| 5 | **note定期購読は審査制** | プレミアム会員（月額500円）への加入と審査が必要 |

## コンサルタントとしての最終推奨

### 結論：「プランC（即日開始）→ プランB（3ヶ月後移行）」の2段階戦略

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  月1          月2          月3          月4          月5          月6
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [============ プランC運用 ============]
  note定期購読 + STORES鑑定書         ↓ データ蓄積・需要検証完了
                                      ↓
                              [==== プランB開発 ====]
                              自社Webアプリ構築   ↓ ローンチ
                                                  ↓
                                          [======= プランB本格運用 =======]
                                          フリーミアム + サブスク + 都度課金

                                          noteは継続（SEO流入チャネルとして）
                                          STORESはWebアプリに統合
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### なぜこの順番か（論理的根拠5点）

**1. リスク最小化の原則**
プランCは固定費がほぼゼロ。「まず売れるかどうか」を検証してから開発投資するのが合理的。
調査データで有料占いコンテンツ利用率は1〜2割。実際のCVRを自分のフォロワーで測定してから次の投資判断をすべき。

**2. 需要の実データが取れる**
プランCの2〜3ヶ月で「どの星座が売れるか」「どの悩み（恋愛/復縁/不倫）が最も反応するか」「価格感度はどこか」の実データが取得できる。
このデータがプランBの機能設計・料金設計を最適化する。

**3. コンテンツ資産の先行蓄積**
プランCで蓄積したnote記事はSEO資産として残る。プランB移行後もGoogle検索流入を生み続ける。
STORESの販売実績・レビューもプランBの信頼性証明に使える。

**4. プランBの手数料優位は規模が出てから効く**
手数料差（note 25% vs Stripe 3.6%）が大きな差額になるのは月売上50万円以上から。
月売上50万円未満のフェーズでは、開発コスト・固定費の方が手数料差より大きいため、プランCの方が利益率が高い。

**5. プランAは「保険」として並行検討**
Instagramサブスク機能は招待制のため確実性が低い。招待が来た場合はプランCと並行して月額400円で開始し、追加の収益チャネルとする。

### 推奨する行動ステップ

| 週 | アクション |
|----|-----------|
| **今週中** | noteプレミアム会員登録（月額500円）→ 定期購読マガジン審査申請 |
| **今週中** | STORESにAI鑑定書の新商品を3つ追加（恋愛相性/復縁/年間恋愛運） |

| **2ヶ月目** | 数字に基づきコンテンツ・価格を調整。AI鑑定書の品質改善 |
| **3ヶ月目** | データを基にプランB（自社Webアプリ）の設計開始 |
| **4〜5ヶ月目** | プランB開発・テスト |
| **6ヶ月目** | プランBローンチ。noteは継続、STORESはWebアプリに統合 |
