#### 価値① 毎朝の限定ストーリーズ「今日のあなたの恋愛運」（毎日 朝7時）

**無料ストーリーズとの違い（これが課金の最大の理由になる）**:

| | 無料（現在の投稿） | サブスク限定（紫リング） |
|--|------------------|----------------------|
| 内容 | 12星座の運勢ランキング（順位のみ） | **あなたの星座だけの恋愛運を深掘り** |
| 深さ | 1位〜12位の一覧 | 恋愛の流れ + 片思い/交際中/フリー別アドバイス |
| アクション | なし | **今日やるべき恋愛アクション1つ** |
| タロット | なし | **今日のタロット1枚引き（カード画像+意味）** |
| JUNの言葉 | なし | **JUNからの2行メッセージ** |
| 保存性 | 24hで消える | **ハイライトに自動保存（見返せる）** |

**5枚構成の詳細**:

```
[1枚目] あなたの星座の恋愛運 ★★★★☆
  「今日の牡羊座は、思い切った行動が吉。
   片思い中→思い切ってDMを。相手は待ってます。
   交際中→些細なことでぶつかりやすい日。深呼吸を。
   フリー→出会いの予感。いつもと違う場所に行ってみて。」

[2枚目] 今日の恋愛タロット
  [AI生成の美しいカード画像]
  「女帝のカード — 愛情に満たされる日。
   受け取ることを恐れないで。」

[3枚目] 今日のラッキー恋愛アクション 🍀
  「好きな人のSNSに"いいね"ではなく、ひとことコメントを。
   たった一言で距離がぐっと縮まります。」
  ※毎日変わる具体的で実行可能な1アクション

[4枚目] JUNからあなたへ 💜
  「片思いがつらい夜もあるよね。
   でも、好きでいられること自体が才能です。
   今日もあなたの恋を応援してます。」
  ※2〜3行の短い言葉。AIが文体を学習して生成→JUN確認

[5枚目] 今日の恋愛運まとめカード
  [保存・スクショ用のデザイン画像]
  星座 / 運勢 / カード名 / ラッキーカラー / アクション を1枚に
  ※「友達に送りたくなる」デザイン → 口コミ拡散
```

**なぜこれに400円の価値があるか**:
- 1日あたり約13円。自販機のジュースより安い
- 競合のゲッターズ飯田（330円）は「総合運+恋愛運+仕事運+金運」を広く浅く → JUNは**恋愛だけを深く**
- 毎朝の「占い→行動→結果」のサイクルが習慣化すると解約できなくなる
- 5枚目のまとめカードがスクショ→友人にシェアされて新規獲得に

## 第3部：noteのコンテンツ設計 — Instagramとの差別化

### 根本的な役割の違い

```
┌──────────────────────┐    ┌──────────────────────┐
│   Instagram サブスク   │    │      note            │
│                        │    │                      │
│  「毎日のお守り」       │    │  「保存版の教科書」   │
│                        │    │                      │
│  ・短い（5枚/60秒）    │    │  ・長い（2000〜5000字）│
│  ・消費型（見て終わり） │    │  ・保存型（何度も読む）│
│  ・感情に寄り添う       │    │  ・知識を深める       │
│  ・「今日どうする？」   │    │  ・「なぜそうなる？」 │
│  ・既存フォロワー専用   │    │  ・Google検索で新規獲得│
│  ・毎日更新             │    │  ・週1〜月数本       │
│  ・400円               │    │  ・500円 + 都度課金   │
└──────────────────────┘    └──────────────────────┘
```

### フォロワーへの説明テンプレート

```
Instagramサブスクの告知:
  「毎朝、あなたの恋愛運が届きます。
   今日やるべき恋愛アクション付き。
   毎日の"お守り"として使ってください💜」

noteの告知:
  「Instagramでは伝えきれない"恋愛占いの深い話"を

   なぜあの人は連絡してこないのか。
   復縁できる人とできない人の違いは何か。
   じっくり読みたい方はnoteへ。」
```

**同じ人が両方入っても「被り感」がない理由**:
- Instagramは「今日の運勢」→ 日替わりで消費するもの
- noteは「恋愛の本質論」→ 何度も読み返すもの
- 例えるなら: Instagram＝毎朝の天気予報、note＝気象学の本

#### 柱2: 有料単品記事（500〜1,500円）— Google検索からフォロワー外を獲得

**これがnoteの真の武器。SEO流入でフォロワー外にリーチする。**

noteの記事はGoogleにインデックスされる。つまり「復縁 占い タロット」「○○座 恋愛運 2026」で検索した人が、JUNさんを知らなくてもnoteの記事にたどり着く。

**SEOを意識した記事タイトル設計**:

| 狙う検索キーワード | 記事タイトル | 価格 | 構成 |
|-------------------|------------|------|------|
| 「復縁 占い タロット」 | **「復縁できる人とできない人の決定的な違い — タロット占い師が500件の相談から見えたこと」** | 800円 | 無料: 3つの違いのうち2つ → 有料: 3つ目+具体的行動+タロットメッセージ |
| 「○○座 恋愛運 2026 下半期」 | **「【2026年下半期】12星座別・恋愛運完全ガイド — 運命の出会い月カレンダー付き」** | 1,200円 | 無料: 3星座分 → 有料: 全12星座+月別カレンダー |
| 「片思い 脈あり 占い」 | **「脈あり？脈なし？ 相手の行動パターンをタロットで読み解く7つのサイン」** | 500円 | 無料: 3つのサイン → 有料: 残り4つ+タロット解釈 |
| 「不倫 占い 結末」 | **「不倫の結末をタロットは知っている — 占い師が語れなかった"その後"の話」** | 1,000円 | 無料: 導入+問題提起 → 有料: 実例パターン+タロット鑑定 |
| 「結婚 時期 占い 当たる」 | **「結婚の時期は占いで分かるのか？ — 800人を鑑定して見えた"法則"」** | 800円 | 無料: 結論と概要 → 有料: 星座別の傾向+具体的時期のヒント |
| 「彼氏 連絡こない 不安」 | **「彼からLINEが来ない夜に読む、占い師からの手紙」** | 300円 | 無料: 前半の寄り添い → 有料: タロットメッセージ+行動アドバイス |

**なぜこの設計が「フォロワー外にも売れる」のか**:

1. **タイトルに検索キーワードを含む** → Googleに引っかかる
2. **無料部分で「この人の文章、信頼できる」と感じさせる** → 有料部分を購入
3. **具体的な数字をタイトルに入れる** → 「500件の相談」「800人を鑑定」「7つのサイン」= 購入率が約2倍（[note公式の分析](https://note.com/notemag/n/nf7284792a3b6)）
4. **無料部分が十分に価値がある** → SNSでシェアされる → さらに流入

## 第4部：2チャネルの告知・プロモーション方法

### Instagramプロフィール

```
🔮 恋愛占い師 JUN｜タロット
💜 サブスクで毎朝あなたの恋愛運をお届け中
📖 note「JUNの恋愛占いノート」→ リンク
✨ 個人鑑定書のお申込みもこちらから
```

### 毎日の自然な告知（最重要）

```
[通常ストーリーズ] 「今日の星座運勢ランキング！1位は射手座🎉」
                    ↓ すぐ隣に
[紫リングの限定ストーリーズ] 射手座の恋愛運詳細+タロット+アクション

→ 一般フォロワーが紫リングをタップ
→ 「サブスク限定です」画面が表示される
→ これが毎日×8万人への自動告知
→ 「気になる…」が蓄積されて、ある日登録する
```

### note記事の拡散

```
月2本の無料記事をInstagramで告知:
  リールやストーリーズで
  「noteに新しい記事を書きました📖
   "彼からLINEが来ない夜に読む、占い師からの手紙"
   無料で読めます → プロフリンク」

→ フォロワーがnoteに流入
→ 無料記事を読む → 「この人の文章いい」
→ 定期購読マガジン or 有料記事を購入
→ さらにGoogle検索から新規読者が流入
→ noteのプロフィールからInstagramをフォロー
→ Instagramサブスクに登録

＝ 双方向の循環が生まれる
```
