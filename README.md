# はじめて注文住宅相談LINE - 静的LP

ビルド不要の静的ランディングページです。`index.html` をブラウザで開くだけで表示できます。

## ファイル構成

```
├── index.html          … メインHTML（全11セクション）
├── assets/
│   ├── style.css       … スタイルシート（CSS変数でデザイントークン管理）
│   └── main.js         … FAQ・ハンバーガーメニュー・スクロールアニメーション
└── README.md           … このファイル
```

## 運用時の差し替え箇所

### 1. LINE リンク（CTA ボタン）
`index.html` 内の `#TODO_LINE_URL` を実際の LINE 友だち追加 URL に置換してください。

```
検索: #TODO_LINE_URL
置換: https://lin.ee/xxxxx（実際のURL）
```

該当箇所: ヘッダーCTA / ヒーローCTA / 最後のCTA（計3箇所）

### 2. プライバシーポリシー・利用規約リンク
フッター内の以下を実際のURLに置換してください。

- `#TODO_PRIVACY_URL` → プライバシーポリシーページURL
- `#TODO_TERMS_URL` → 利用規約ページURL

### 3. 文言の変更
`index.html` 内のテキストを直接編集してください。
セクションはHTMLコメントで区切られています（例: `<!-- ========== 2) ヒーロー ========== -->`）。

### 4. カラー・フォントの変更
`assets/style.css` 冒頭の `:root` 内の CSS 変数を変更してください。

| 変数 | 用途 | デフォルト |
|------|------|-----------|
| `--color-primary` | メインカラー（LINEグリーン） | `#06C755` |
| `--color-primary-dark` | ホバー時のボタン色 | `#05a648` |
| `--font-family` | フォントファミリー | Noto Sans JP + システムフォント |

## 確認ポイント

- **スマホ表示**: 960px以下で1カラムレイアウトに切り替わります
- **FAQ動作**: クリックでアコーディオン開閉（`<details>`/`<summary>`使用）
- **CTAリンク**: `#TODO_LINE_URL` を検索し、全箇所が正しいURLに差し替え済みか確認
- **スクロールアニメーション**: カード・ステップ等がスクロールでフェードイン表示
